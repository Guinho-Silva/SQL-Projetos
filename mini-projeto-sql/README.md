# Banco de Dados – Projeto de Estudo SQL

Este repositório contém um banco de dados desenvolvido com o objetivo de **estudo e prática de SQL**, modelagem de dados e consultas relacionais.

O projeto faz parte do meu processo de aprendizado na área de **dados / desenvolvimento backend**, utilizando banco de dados relacional (SQLite/XAMPP).

---

## Objetivo do Projeto

- Praticar conceitos fundamentais de **SQL**
- Treinar:
  - `SELECT`, `WHERE`, `INSERT`, `UPDATE`, `DELETE`
  - **modelagem de dados** e relacionamento entre entidades

---

##  Estrutura do Projeto

- O arquivo [mini_projeto.sql](/mini_projeto.sql) contém todos os dados que eu usei para práticar na linguagem sql, desde a criação do banco até sua manipulação.
- Os demais arquivos, sendo eles:
  - [ddl.sql](/ddl.sql) — *Data Definition Language* (criação das tabelas)
  - [dml.sql](/dml.sql) — *Data Manipulation Language* (inserção e alteração de dados)
  - [dql.sql](/dql.sql)  — *Data Query Language* (consultas)
</br>
  <p>Mostram a execução de cada etapa do processo de construção e uso do banco de dados.</p>
